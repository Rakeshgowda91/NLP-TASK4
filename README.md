
# NLP Task 4 - BERT Fine-Tuning

This project demonstrates fine-tuning of a BERT model on a Kaggle dataset as part of an NLP internship assignment.

## Objective

To preprocess text data and apply BERT for classification using modern NLP techniques.

## Features

* Data preprocessing and cleaning
* Tokenization using BERT tokenizer
* Model training and evaluation
* Performance analysis

## Technologies Used

* Python
* Jupyter Notebook
* Transformers (Hugging Face)
* Scikit-learn

## File

* bert_finetuning.ipynb
